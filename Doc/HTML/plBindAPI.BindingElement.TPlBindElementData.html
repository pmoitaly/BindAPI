Unit

plBindAPI.BindingElement

Declaration

type TPlBindElementData = class(TObject)

Description

An element data contains source or target atomic info.

Two instances of this class are stored in the binder that contain informations about bindong's source & target.

Do not use this class directly. <see cref=”TPlBinder” \> manages all necessary instances of this class.
