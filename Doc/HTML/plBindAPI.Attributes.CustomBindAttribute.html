Declaration

type CustomBindAttribute = class(TCustomAttribute)

Description

Ancestor class of all BindApi attributes

BindApi attributes contain general binding instructions. They can be used to create data-binding scenarios where changes in one object's properties are automatically reflected in another.

property IsEnabled: Boolean read FIsEnabled;

Enable or disable the attribute.

An attribute is processed only when its IsEnabled property is true. You can enable or disable any attribute at design time. If you omits this parameter, BindApi assumes that it is True, so you can write a bit cleaner code.

<value> Boolean </value>

property TargetClassAlias: string read FTargetClassAlias;

An alias name for the target class.

Target class alias is an alias of the class where we search for the field/property/method to be bound to an attribute owner (or source) elmement. Use it, for instance, in inheritable classes to ensure that you can place correct attributes in superclasses.

property TargetClassName: string read FTargetClassName;

The target class' name.

Target class is the class involved in the binding operation. Note that the source class is always the class that contains the attribute.

<value> string </value>
