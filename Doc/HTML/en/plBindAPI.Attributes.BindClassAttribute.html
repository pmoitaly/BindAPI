Declaration

type BindClassAttribute = class(CustomBindAttribute)

Description

A Class attribute describing the target class of bind action.

This attribute tells that the following class must contains the bind information. The constructor has many signatures, supporting different options. However, the only mandatory parameter is ATargetClassName.

<example> 

type
 [BindClass(True, 'MyBindTargetClass')]
 TMyForm = class(TForm)
  public

  ...

</example>

Initializes a new instance of the BindClassAttribute class with enabled status, target class name, target class alias, and an optional default class flag.

Set Enabled to False to skip the binding. This could be usefull at design time, to test the behaviour of some specific binding.

property IsDefault: Boolean Read FIsDefault;

Gets a value indicating whether the target class is the default class for this binding.

If True, any binding that does not specify the Target class will refer to this one. This param will be removed in future. Please, use BindDefaultClass instead.
