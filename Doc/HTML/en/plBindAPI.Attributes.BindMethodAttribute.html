Unit

plBindAPI.Attributes

Declaration

type BindMethodAttribute = class(CustomBindAttribute)

Description

Ancestor class for attributes containing information about a single bind of methods and events.

Use this attribute to replace a method or an event indentified by SourceMethidName with the method called NewMethidName in the Target class. <note type="warning">If the Source's method is already defined, it is replaced by the new method and no info is kept. So you will not able to restore it.</note>

constructor Create(const Enabled: Boolean; const AMethodQName, ANewMethodName: string); overload;

Initializes a new instance of the BindMethodAttribute class with enabled status, qualified name of the source method, and name of the new method.

<example> 

type
   {...}
   [BindDefaultClass(True, 'MyBindTargetClass')]
   [BindMethod(True, 'MyMethod', 'NewMethod')]
   TMyForm := class(TForm)
   public
     MyMethod:  TNotifyEvent;
   {...}

or 

type
   [BindDefaultClass(True, 'MyBindTargetClass')]
   [BindMethod(True, 'Button1.OnClick', 'NewEventHandler')

] TMyForm := class(TForm) Button1: TButton; {...} ) </example>
