type BindMemberAttribute = class(CustomBindMemberAttribute)

Description

Requires a bi-directional bindig for the involved member.

When this attribute is encountered, a bindig is created from the targets' member to then source's member. See CustomBindMemberAttribute form methods, properties and examples.
