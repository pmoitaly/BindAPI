Unit

plBindAPI.BindManagement

Declaration

type TPlBindManager = class(TInterfacedObject)

Description

Manages binding operations and provides utilities for deferred and immediate bindings.

Using TPlBindManager is the easiest and fastest way to manage an application based on the BindAPI framework.

Use this class to parse the attributes of a class and manage BindAPI ones' without create any instance of other objects.

It is a completely static class that manacontains an instance of TPlAutoBinder, to which it sends information read from the attributes of a class (usually a TForm, TFrame or TComponent descendant).

<example> The following example comes from the form of BindApiSimpleDemo. 

procedure TfrmBindApiSimpleDemo.FormCreate(Sender: TObject);
  begin
    {Remember: if the bound class is not a singleton, the binder is
     responsible of its destruction}
   TplBindManager.Bind(Self);
  end;

  procedure TfrmBindApiSimpleDemo.FormDestroy(Sender: TObject);
  begin
    TplBindManager.Unbind(Self);
  end;

</example>
